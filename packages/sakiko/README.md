## zh-CN

这个包是 sakiko 项目的核心框架包体，作为聊天机器人应用开发的基础框架部分，整合项目中的各个模块模块，作为开发者和事件系统之间交互的桥梁。

## en-US

> This project is a part of the chatbot framework "[Sakiko](https://github.com/togawa-dev/sakiko)".

This package is the core framework package of the sakiko project. As the foundational framework part for chatbot application development, it integrates various modules in the project and serves as a bridge between developers and the event system.

The project name Sakiko comes from the band _Ave Mujica_'s keyboardist `Togawa Sakiko` a.k.a _`Oblivionis`_ in the cross-media project _BanG Dream!_.

This page is not the homepage of the sakiko framework project. For more information, see the [main repository](https://github.com/togawa-dev/sakiko) or the [documentation](https://togawa-dev.github.io/docs/).

# @togawa-dev/umiri

## zh-CN

> 这个项目是聊天机器人框架 [Sakiko](https://github.com/togawa-dev/sakiko) 的一部分。<br>

umiri 是一个基于类型系统和优先级机制派发事件的本地事件总线，专为构建简洁高效的聊天机器人应用设计。

项目名称 umiri 来自跨媒体企划 _BanG Dream!_ 中乐队 _Ave Mujica_ 的贝斯手`八幡海铃`（Yahata **Umiri** a.k.a _Timoris_）

umiri 仅作为 sakiko 框架内部的事件总线实现，对于独立的基于类型系统的事件总线实现，见 [@togawa-dev/taki](https://github.com/togawa-dev/taki) 。

## en-US

> This project is a part of the chatbot framework "[Sakiko](https://github.com/togawa-dev/sakiko)".

Umiri is a local event bus based on a type system and priority mechanism, designed for building simple and efficient chatbot applications.

The project name Umiri comes from the band _Ave Mujica_'s bassist `Yahata Umiri` a.k.a _`Timoris`_ in the cross-media project _BanG Dream!_.

Umiri is only used as the event bus implementation inside the sakiko framework. For an independent type-system-based event bus implementation, see [@togawa-dev/taki](https://github.com/togawa-dev/taki).

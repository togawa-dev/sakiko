# @togawa-dev/uika

## zh-CN

> 这个项目是聊天机器人框架 [Sakiko](https://github.com/togawa-dev/sakiko) 的一部分。<br>

uika 是 sakiko 的高级功能拓展包，其中的各个分包为 sakiko 提供了聊天机器人应用开发中各种常用的基础功能模块。

项目名称 uika 来自跨媒体企划 _BanG Dream!_ 中乐队 _Ave Mujica_ 的主唱 & 吉他手`三角初华`（Misumi **Uika** a.k.a _Doloris_）

## en-US

> This project is a part of the chatbot framework "[Sakiko](https://github.com/togawa-dev/sakiko)".

Uika is an advanced feature extension package for sakiko, with various sub-packages providing common foundational modules for chatbot application development within sakiko.

The project name Uika comes from the band _Ave Mujica_'s vocalist & guitarist `Misumi Uika` a.k.a _`Doloris`_ in the cross-media project _BanG Dream!_.
